# 背景图片添加指南

## 如何添加背景图片

### 方法1：使用本地图片文件

1. **准备图片**
   - 准备一张森林主题的图片（建议尺寸：1920x1080 或更大）
   - 建议使用虚化/模糊效果的自然森林图片
   - 图片格式：JPG、PNG、WebP 都可以

2. **创建图片目录**
   ```
   大手町の森_環境センサー可視化/
   ├── images/
   │   └── forest-background.jpg  (您的图片)
   ├── index.html
   └── ...
   ```

3. **修改代码**
   
   打开 `index.html` 文件，找到以下代码（大约在第36-50行）：
   
   ```css
   background: 
     /* 半透明白色层 */
     linear-gradient(...),
     /* 森林背景图片 - 取消下面这行的注释并修改图片路径 */
     /* url('images/forest-background.jpg'), */
     /* 备用渐变背景 */
     ...
   ```
   
   取消注释 `url('images/forest-background.jpg')` 这一行：
   
   ```css
   background: 
     linear-gradient(...),
     url('images/forest-background.jpg'),  /* 取消注释这一行 */
     ...
   ```
   
   如果您的图片文件名或路径不同，请相应修改。

### 方法2：使用在线图片URL

如果您想使用网上的图片，可以直接使用图片的URL：

```css
background: 
  linear-gradient(...),
  url('https://example.com/path/to/forest-image.jpg'),  /* 使用在线图片 */
  ...
```

### 方法3：使用Base64编码图片

对于小图片，可以转换为Base64编码直接嵌入CSS中（不推荐用于大图片，会增加文件大小）：

```css
background: 
  linear-gradient(...),
  url('data:image/jpeg;base64,/9j/4AAQSkZJRg...'),  /* Base64编码 */
  ...
```

## 图片建议

### 推荐图片特性：
- **主题**：自然森林、绿色植物、树木
- **风格**：柔和、虚化、自然光
- **色调**：绿色系，与整体设计协调
- **尺寸**：建议至少 1920x1080 像素
- **文件大小**：优化后建议小于 500KB，保证加载速度

### 图片处理建议：
- 使用图片编辑软件（如 Photoshop、GIMP）添加轻微的高斯模糊
- 降低对比度，使其作为背景时不会干扰前景内容
- 调整色调，使其偏向绿色系
- 可以使用透明度或遮罩层进一步柔化效果

## 注意事项

1. **版权**：确保您使用的图片有合法的使用权
2. **性能**：大图片可能影响加载速度，建议优化图片大小
3. **兼容性**：使用 `background-attachment: fixed` 在某些移动设备上可能不工作，这是正常的
4. **备用方案**：代码中已包含渐变背景作为备用，即使图片加载失败也会显示渐变背景

## 测试

添加图片后，在浏览器中打开 `index.html` 查看效果。如果图片没有显示：
- 检查文件路径是否正确
- 检查图片文件是否存在
- 打开浏览器开发者工具（F12）查看是否有错误信息
- 确保图片文件格式正确（JPG、PNG等）
